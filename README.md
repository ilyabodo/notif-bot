# notif-bot
Notification bot for discord that detects changes in website html <br>
v0.1
<br>
##### Bot usage:  
```!!notif_add https://example.com```<br>
-> Adds new site to be monitored. User will be @ mentioned in the channel where the command is given
<br>
<br>
```!!notif_list```<br>
-> Displays list of all of user's monitored sites
<br>
<br>
```!!notif_remove https://example.com``` <br>
-> Removes monitoring from example.com
<br>
<br>
```!!notif_remove_all``` <br>
-> Removes all of current users sites
